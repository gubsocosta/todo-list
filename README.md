# Todo List

## Description
This project is a **Todo List** developed during the course **"Vue JS 3 Complete with Composition API,
Vuex & Vue Router"**. The application allows users to manage their daily tasks efficiently with basic CRUD functionality..

## Technologies Used
- **Vue 3**: Progressive JavaScript framework for building user interfaces.
- **Vue CDN**: Vue is used directly via CDN, without the need for build tools.

## Features
- **Add Todo**: Easily add new tasks through a form. 
- **View Todos**: Display a list of all tasks with their status. 
- **Mark Todo as Done**: Mark tasks as completed with a single click. 
- **Clear Todos**: Remove all tasks at once to reset the list. 
- **Real-Time Updates**: Automatically updates the list without page reloads.
- **Persisted State**: Todos are stored locally using the browser's local storage, ensuring data is retained even after refreshing the page.

## Project Structure
The project was developed using Vue's CDN, so there is no complex folder structure. The code is organized in a single HTML file,
with Vue logic separated into sections for better understanding.
